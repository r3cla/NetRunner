- Toggleable Win95-style windows ‚ÄĒ sections open/close via the nav bar
- Cyberpunk terminal aesthetic ‚ÄĒ scanlines, glitch animation, ASCII art
- Fully responsive ‚ÄĒ mobile, tablet, and desktop layouts
- Accessible ‚ÄĒ semantic HTML, ARIA labels, keyboard navigation (Alt+1-4)
- Reduced-motion support via `prefers-reduced-motion`
- High-contrast mode support via `prefers-contrast`
- Print stylesheet included
